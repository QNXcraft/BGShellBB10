# GitHub Actions CI Badges

Add these badges to your README.md file to show build status:

## Badge Markdown

### Build Status
```markdown
![Build](https://github.com/YOUR_USERNAME/BGShellBB10/actions/workflows/build.yml/badge.svg)
```

### Docker Build Status
```markdown
![Docker Build](https://github.com/YOUR_USERNAME/BGShellBB10/actions/workflows/docker-build.yml/badge.svg)
```

### Latest Release
```markdown
![GitHub release (latest by date)](https://img.shields.io/github/v/release/YOUR_USERNAME/BGShellBB10)
```

### Downloads
```markdown
![GitHub all releases](https://img.shields.io/github/downloads/YOUR_USERNAME/BGShellBB10/total)
```

### License
```markdown
![License](https://img.shields.io/github/license/YOUR_USERNAME/BGShellBB10)
```

## Complete Badge Section Example

Replace `YOUR_USERNAME` with your actual GitHub username:

```markdown
# BGShellBB10

![Build](https://github.com/YOUR_USERNAME/BGShellBB10/actions/workflows/build.yml/badge.svg)
![Docker Build](https://github.com/YOUR_USERNAME/BGShellBB10/actions/workflows/docker-build.yml/badge.svg)
![GitHub release (latest by date)](https://img.shields.io/github/v/release/YOUR_USERNAME/BGShellBB10)
![GitHub all releases](https://img.shields.io/github/downloads/YOUR_USERNAME/BGShellBB10/total)
![License](https://img.shields.io/github/license/YOUR_USERNAME/BGShellBB10)

Provides you with an ordinary terminal window, i.e. sh running on your BlackBerry BB10...
```

## Usage

1. Open your `README` or `README.md` file
2. Copy the badge markdown from above
3. Replace `YOUR_USERNAME` with your GitHub username
4. Paste at the top of your README file

The badges will update automatically based on your GitHub Actions runs and releases!
